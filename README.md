# Modeling-and-Simulation-of-Electric-Vehicle-Propulsion-System-using-BLDC-Motor
The project aims at developing and analyzing an electric vehicle (EV) drive train using MATLAB R2024a and Simulink software package. The project will involve simulation of the whole EV propulsion system and performance evaluation.

The project will be based on a BLDC motor which is popular in the electric car industry because of its high efficiency, small size, low maintenance and speed-torque capabilities. The EV propulsion system contains a lithium-ion battery, three-phase voltage source inverter, electronic commutation with the help of Hall sensors, current control loops and PI-based speed control loop.

A battery works as an energy source providing the DC voltage input to the inverter block. The three-phase voltage source inverter converts DC voltage into controlled three-phase voltages required to rotate the BLDC motor. Hall sensors control the rotor position and provide information needed for correct electronic commutation. PI control algorithm controls the motor speed by manipulating the switching signal provided to the inverter according to the reference speed value.

The simulation model makes it possible to perform an in-depth study of the following parameters, such as motor speed, electromagnetic torque, stator current, back electromotive force (Back-EMF), battery voltage, current usage, and state of charge (SOC). The evaluation of these parameters makes it possible to assess the efficiency and stability of the operation of the vehicle propulsion system at different speeds and under varying loads.

This work gives one an opportunity to get some insights into drive systems of electric vehicles and the practical use of power electronics, motor drives, control systems, and batteries in electric vehicles.
